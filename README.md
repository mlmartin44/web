How to Use the Website
Main Page
Navigation:
The main page provides navigation links to different sections:

View Data: View all existing users.

Add New Data: Add a new user.

Modify Existing Data: Update details for an existing user.

View Data Page
View Users:
On the View Data page, you'll see a list of all users stored in the database.

Add New Data Page
Add New User:
On the Add New Data page, fill in the form with the required information (name, email), then click the submit button to create a new user. This will save the data to the database.

Modify Existing Data Page
Modify User Data:
To modify an existing user's information, first, access the Mod4 UserDB database to find the user ID.
Then, navigate to the Modify Existing Data page, where you'll enter the user ID and the new data (name, email) to update the user's information.

Testing the Website Locally
Open the website using VS Code with the Live Server extension.

Test the navigation by clicking between pages (Home, View Data, Add Data, Modify Data).

Ensure that the form on the Add Data page works by adding a new user.

Check that the Modify Data page allows you to edit and update a user's information.

Validate that the View Data page displays the updated list of users after adding or modifying data.
